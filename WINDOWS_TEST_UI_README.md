## 🚀 3 步快速启动

### 步骤 1: 启动 API 服务器

```bash
cargo run --bin rikkahub-server
```

等待看到：`服务器运行在 http://0.0.0.0:3000`

### 步骤 2: 构建桌面客户端

```bash
.\build-windows.bat
```

脚本会自动检查环境、清理并构建。

### 步骤 3: 运行测试界面

```bash
.\run-test.bat
```

或者直接运行：`.\target\release\rikkahub.exe`

## 🔧 故障排除

### 问题: 编译失败
```bash
# 解决方案
rustup update
cargo clean
cargo build --release --bin rikkahub
```

### 问题: 窗口无法显示
- 确保安装了 Visual Studio Build Tools
- 检查是否有最新的显卡驱动
- 尝试以管理员权限运行

### 问题: 连接失败
- 确保 API 服务器正在运行
- 检查端口 3000 是否可用
- 查看终端日志获取错误信息
